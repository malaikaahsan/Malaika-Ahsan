## 🚀 Featured Projects

### 🔐 [IoT Attack Classification ML Model](https://github.com/malaikaahsan/IoT-Attack-Classification-ML-Model)
Multi-class classification of 10 IoT cyberattack types (DDoS, Ransomware, MITM, etc.) on a dataset of 211,000+ network traffic records. Built a full 5-phase ML pipeline covering EDA, feature engineering, LightGBM modeling, class imbalance handling, and feature selection.
`Python` `LightGBM` `XGBoost` `scikit-learn` `Pandas`

---

### ☕ [Cafe Website — Full Stack](https://github.com/malaikaahsan/Cafe-Website-Node.js-MySQL-Full-Stack)
A full-stack cafe management website with customer menu browsing and an admin panel for management. Built with a REST-style Node.js backend connected to a MySQL database.
`Node.js` `MySQL` `HTML` `CSS` `JavaScript`

`Node.js` `HTML` `CSS` `JavaScript`
